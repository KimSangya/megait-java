### Product(상품) 테이블 만들기

필드

​				---- 상품 번호 (pd_no). 우선키. 자동증가

​				---- 상품 이름 (pd_name). 255바이트. 가변문자열. 누락X

​				---- 상품 가격 (pd_price). 누락X

상품 레코드 5~6개 추가



```mariadb
CREATE TABLE product (

pd_no INT PRIMARY KEY AUTO_INCREMENT,

pd_name CHAR(255) NOT NULL,

pd_price INT DEFAULT 0 NOT NULL

);
```

```mariadb
INSERT INTO product VALUES(NULL, 'flower', 1000);
INSERT INTO product VALUES(NULL, 'computer', 10000);
INSERT INTO product VALUES(NULL, 'cracker', 500);
INSERT INTO product VALUES(NULL, 'watermelon', 2500);
INSERT INTO product VALUES(NULL, 'bottle', 2000);
```

### cart(장바구니) 테이블 만들기

필드 ------- 장바구니 번호(ct_no). 우선키.자동증가

​		-------- 회원번호(mem_no). 상품을 찜한 회원 정보

​		-------- 상품 번호(pd_no). 찜된 상품

상품 레코드 추가

​				-- 2번 회원이 1,4번 상품을 찜함

​				-- 3번 회원이 1번 상품을 찜함.

​				-- 2번 회원이 2번 상품을 찜함.

ct_no mem_no pd_no

1               2              1

2               2              4

3               3              1

4               2               2

```mariadb
CREATE TABLE cart (

ct_no INT PRIMARY KEY AUTO_INCREMENT,

mem_no INT DEFAULT 0 NOT NULL,

pd_no INT DEFAULT 0 NOT NULL

);
```

```mariadb
INSERT INTO cart VALUES (NULL, 2 , 1), (NULL, 2 , 4);
INSERT INTO cart VALUES (NULL, 3 , 1);
INSERT INTO cart VALUES (NULL, 2 , 2);
```

cart 테이블에 외래키 적용하기.

```mariadb
ALTER TABLE cart
ADD CONSTRAINT cart_pd
FOREIGN KEY(pd_no)
REFERENCES product(pd_no);
```
