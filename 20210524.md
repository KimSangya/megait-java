# 05.24 과제

과제 1번은 못 풀겠어서 2번만 했습니다..!!

```java
package com.day23.quiz;
import java.util.TreeSet;

public class Quiz02 {
	public static void main(String[] args) {
		//treeset를 사용하면 5줄로 끝낼수있음.
		//로또 생성기 : 
		// 중복없이 1 ~ 45 중 6개의 숫자 뽑기
		// 오름차순 정렬
		// 결과 출력
		TreeSet<Integer> lottoNums = new TreeSet<Integer>();
		for(int i = 0; lottoNums.size() < 6; ++i) {
			lottoNums.add((int)(Math.random() * 45) +1 );
		}
		System.out.println("로또 결과는 : " + lottoNums);
	}
}
```
