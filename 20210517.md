# 익명 클래스

익명클래스 = 무명클래스.

익명의 성질을 가진 클래스라고합니다.

```java
public class Test01{
    private int num = 0;
    public int getNum(){
        return this.num;
    }
    public void setNum(int num){
        this.num = num;
    }
}
```

맴버변수 num를 접근 할 수 없지만, getter, setter로 접근 할 수 있게 만들었습니다.

```java
Test01 a = new Test01();
```

평상시에는 만들때는 이렇게 만들어서 생성하는게 보통이다.

그렇지만 익명 클래스는, 

```java
Test01 a = new Test01(){
    public int num = 0;
    @Override
    public int getNum(){
        return this.num;
    }
}
```

이렇게 생성을 하여, 어떠한 클래스가 부모 클래스로 부터 상속 받은 것인 클래스이지만, 이름이 없어서

생성자를 선언할수도 없고, 이름도 없어서 익명 클래스라고 불린다.
