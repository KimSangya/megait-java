# 5.7일 과제

### 1번

 * 클래스 : Rectangle

 * - 필드(멤버변수) : base, height

 * - 메소드 

 * 1) setData() : 두 정수를 인자값으로 받아, 객체의 base, height에 각각 저장

 * 2) getArea() : 넓이를 리턴

 * 3) getCircum() : 둘레를 리턴

 * 

 * 메인 클래스 : Quiz01

 * - Rectangle 객체 1개 생성 

 * - JOptionPane을 사용하여 밑변, 높이를 입력 받고, 

 * Rectangle 객체에 저장 (setData() 사용)

 * - 이 사각형 객체의 넓이와 둘레를 메소드를 사용하여 출력 (getArea(), getCircum())

     **Quiz01**

```java
public class Quiz01 {
	public static void main(String[] args) {
		String a,b; 
		int c,d;
		Rectangle ab = new Rectangle();
		a = JOptionPane.showInputDialog("정수를 입력하세요.");
		b = JOptionPane.showInputDialog("다른 정수를 입력하세요.");
		c = Integer.parseInt(a);
		d = Integer.parseInt(b);
		ab.setData(c, d);
		
		JOptionPane.showMessageDialog(null, "넓이는 : " + ab.getArea());
		JOptionPane.showMessageDialog(null, "둘레는 : " + ab.getCricum());
		
	}
}
```

​			**Rectangle**

```java
public class Rectangle {
	int base, height;
	void setData(int a, int b){
		base = a;
		height = b;
	}
	int getArea(){
		return base*height;
	}
	int getCricum() {
		return (base*2)+(height*2);
	}
}
```

# 2번

 * 클래스 : MyMath

 * - 필드 : 없음

 * - 메소드 

 * 1) getTotal()	: 정수 1개를 인자값으로 받고, 1 부터 그 수까지의 모든 합을 return

 * e.g. 인자값 : 5 -> 리턴값 : 10

 * 인자값 : 3 -> 리턴값 : 6

 * 2) getGugudan() : 정수 1개를 인자값으로 받고, 해당 단을 String형태로 return 

 * 

 * 메인 클래스 : Quiz02

 * 메뉴 : 

 * 1. 총합 구하기

 * 2. 구구단 보기

 * 3. 종료하기

 * - 1번 선택 시 : 정수 1개를 입력 받아 1 ~ 정수까지의 합 출력

 * - 2번 선택 시 : 정수 1개를 입력 받아 해당 단 출력

 * - 3번 선택 시 : 프로그램 종료 

 * (3번을 선택해야 메뉴창 띄우기를 반복 종료함.)

   ​	**Quiz02**

```java
public class Quiz02 {
	public static void main(String[] args) {
		String b;
		int c;
		MyMath a = new MyMath();
		String menu = "1. 총합 구하기 \n"
					+ "2. 구구단 보기 \n"
					+ "3. 종료하기";
		// 방 개수 입력 받기
		
		// 
		
		while(true) {
			
			String select = JOptionPane.showInputDialog(menu);
		
			if("1".equals(select)) {
				
				b = JOptionPane.showInputDialog(null, "총합을 구하겠습니다. 아무 정수를 집어 넣어 주십시오.");
				c = Integer.parseInt(b);
				JOptionPane.showMessageDialog(null, "총합입니다. \n" + a.getTotal(c));
				
			}
			else if("2".equals(select)) {
				b = JOptionPane.showInputDialog(null, "구구단을 합니다. 정수를 하나 집어넣어주십시오");
				c = Integer.parseInt(b);
				JOptionPane.showMessageDialog(null, "총합입니다." + a.getGugudan(c));
			}
			else if("3".equals(select)) {
				JOptionPane.showMessageDialog(null, "프로그램을 종료합니다.");
				break;
			}
		}
	}
}
```

​		**MyMath**

```java
public class MyMath {
	
	int num;
	int getTotal(int a){
		for(int i = 1; i <= a; ++i) {
			num=+i;
		}
		return num;
	}
	String getGugudan(int a){
		String[] b = new String[9]; 
		for(int i = 1; i <= 9; ++i) {
			b[i] = (i + " X " + a + " = " + i*a);
		}
		JOptionPane.showMessageDialog(null,Arrays.toString(b));
		return b[i];
		
	}
}
```
